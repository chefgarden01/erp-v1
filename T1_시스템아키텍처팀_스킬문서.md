# T1. 시스템 아키텍처팀 스킬 문서

## 팀 역할: PDCA - Plan (계획)

## 담당 모듈
- 마스터룰 설계
- 기초정보 (코드생성)
- 사용자계정 관리

---

## 필수 스킬

### 1. 마스터코드 설계
- 코드 체계 자동생성 엔진 설계
- 코드 배열 규칙 수립 및 추천
- 상품관리 마스터코드 (마스터상품, 판매상품, SKU상품, 매핑관리)
- 자재관리 마스터코드 (포장자재, 소모품, 비품, 시설/기구)
- 거래처관리코드 (매출/매입/역발행)
- 비용관리코드, 농장현황코드, 직원코드, 물류관리코드 등

### 2. QR코드 시스템
- 전체 코드 QR코드 이미지 자동생성
- 스마트폰 QR인식 → 해당 메뉴/상품 자동이동
- QR북 기반 입력담당자 퀵메뉴 이동

### 3. 사용자 계정/권한 설계
- 계정 생성/사용/정지 기능
- 접근권한 그룹: 경영그룹, 사용자그룹, 농장그룹, 물류현장그룹, 거래처그룹
- 메뉴설정, 퀵메뉴설정

### 4. UI/UX 아키텍처
- PC웹 (크롬/엣지 최적화)
- 앱 버전 (비공개 배포)
- 사용자별 퀵메뉴 아이콘 자동생성
- 캄보디아어 병행표기 설계

### 5. 데이터 아키텍처
- DB 설계 (대량데이터 처리)
- 백엔드 설계 (빠른 속도)
- 입력/수정 이력관리 추적 체계
- 데이터 보안관리 정책

### 6. 출력 체계
- 모든 화면 프린트 출력 기능
- 출력 양식 설정 및 추천
