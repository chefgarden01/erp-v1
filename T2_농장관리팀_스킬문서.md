# T2. 농장관리팀 스킬 문서

## 팀 역할: PDCA - Do (실행)

## 담당 모듈
- 농장관리 전체
- 생육상태 분석 (AI 연동)
- 농장손익관리

---

## 필수 스킬

### 1. 하우스 현황 관리
- 종합 대시보드 구현 (모종, 정식, 생육, 수확, 휴경, 예정, 신규)
- 상세 조회 (작물별, 기간별 수확일정, 예상물량, 상태관리)
- 하우스별 작물 상세관리 (신규등록, 코드매핑, 기본값/변수값)

### 2. 수확 관리
- 하우스별 수확기록 및 잔량기록
- 수동등록 + QR코드 자동입력
- 출고예약 및 확정 (수확량, 출고실행, 재고현황)

### 3. 수확일정 및 인원관리
- 하우스별/작물별 수확일정 등록
- 인원배정 등록, 예상수확량 등록
- 현장근무자 TO DO 생성/체크/결과확인

### 4. 비용관리
- 종자, 모종, 농자재, 수도광열비, 유지보수비 입력

### 5. 분석 관리
- 시세 자동분석 및 보고서 생성 (API 연동)
- 생산성 관리 (하우스별, 인원별, 작물별)
- 농장손익관리 (자동 손익보고서)

### 6. AI 연동
- AI 작물추천 (KAMIS API + AI 학습 엔진)
- 작물 시뮬레이션 (계획 vs 실적 비교)
- 생육상태 자동분석 (이미지 → AI 분석 → 리포트)
  - 생육상태 판정, 수확예상시점, 문제발생/처방/관리방향
  - 결과값 입력 후 학습 업데이트

---

## 연동 API
- KAMIS API (시세 분석)
- AI 이미지 분석 API (생육 분석)
- AI 학습 엔진 API (작물 추천)

## 산출물
- 하우스 현황 대시보드
- 수확관리 기록부
- 농장손익 보고서
- AI 생육분석 리포트
- 작물추천 보고서
