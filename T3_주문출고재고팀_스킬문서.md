# T3. 주문출고재고팀 스킬 문서

## 팀 역할: PDCA - Do (실행)

## 담당 모듈
- 주문수집 및 집계
- 출고관리
- 재고관리 (상품/비상품)

---

## 필수 스킬

### 1. 주문수집
- 판매처별 주문수집 (엑셀등록, 수동등록, OCR등록)
- 중복등록 방지, 취소/변경 기능
- 자동스케줄러 주문수집 (업데이트 버전)

### 2. 주문집계
- 주문집계 대시보드 (주문건수, 박스건수, 소분건수, 출고존별)
- 주문집계 조회 (마스터상품, SKU, 채널별, 판매처별, 입고센터별)
- 쿠팡 주문리스트 엑셀 자동화
- 주문현황 전체/상세/시간대별 조회

### 3. 추가주문 관리
- 품목별 담당자 추가주문 예상량 입력
- 마감시간 자동리셋, 정확도 분석

### 4. 재고관리
- 상품재고등록 (QR코드, 입고이력코드, OCR, 수동)
- SKU상품 사전작업 임시재고관리
- 비상품 재고관리 (포장자재, 라벨지, 소모품 등)
- 부족/적정/과잉 재고 자동리포트 및 알림

### 5. 생산관리
- SKU별 생산인력배정, 작업시간, 마감시간
- 생산현황 진도율 조회

### 6. 입고예약
- 긴급입고, 사전확보 입고, 조건입고(타겟가)
- 과잉재고 자동경고

### 7. 시세 및 시뮬레이션
- 관심상품 시세조회 (KAMIS, 가락시장 API)
- 공급가 손익시뮬레이션

### 8. 분석
- 생산성 분석 (주문건수, 매출액, 근무시간, 효율)

---

## 연동 API
- KAMIS API, 가락시장 API (시세 조회)
- OCR 엔진 (주문서/재고조사표 인식)

## 산출물
- 주문집계 대시보드
- 재고현황 리포트
- 부족/과잉 재고 알림 보고서
- 생산성 분석 보고서
