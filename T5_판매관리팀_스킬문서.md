# T5. 판매관리팀 스킬 문서

## 팀 역할: PDCA - Do (실행)

## 담당 모듈
- 판매관리1 (오프라인 사업부)
- 판매관리2 (온라인 사업부)

---

## 필수 스킬

### 1. 오프라인 사업부
- 판매처별 견적상품리스트 업로드 (기존/신규)
- 견적서 작성 및 공유 (삼성, 쿠팡, 현대, 차별화)
- 관심품목 시세 조회 (KAMIS API)
- 판매손익 시뮬레이션 (견적가, 예상수량, 매입가, 장려금%, 인건비%, 배송비%, 기타비용)
- 계약 확정 품목 관리
- 견적서 조회/다운로드
- 거래처 자동송신 (연락처, 이메일, 카카오톡)

### 2. 온라인 사업부
- 주문매출 조회
- SKU별 매입가적용 (입고관리 데이터 연동 → 수정 → 확정)
- 매입가적용 규칙 (농장상품/비농장상품/온라인전용 구분)
- 매출이익 조회 (스토어별, 판매처별, 채널별)
- 판매상품 관리 (상세페이지, 썸네일, 스토어, 배송타입, 직영/위탁)

### 3. 판매가 관리
- 판매가 정보수집 및 경쟁사 비교 (API/크롤링)
- 판매가 다운로드 → 사방넷 업로드 → 스토어 전송
- 판매가 조회 (전체, SKU별, 배송타입별, 출고방식별)

### 4. 판매손익 시뮬레이션
- 마스터상품별/SKU별 마진설계 테이블
- 항목: 판매가, 할인율, 수량, 스토어수수료, 배송비, 인건비, 포장자재, 마케팅비, 간접비, 마진

### 5. 손익 조회
- 가마감 손익 (주문수집 + 매입가 + 비용 + 수수료 + 배송비)
- 최종 손익 (데이터 수정/확정)

### 6. 회계 연동
- 회계용 매출입력 (스토어별 카드/현금/기타)
- 회계용 매입입력 (수수료 계산서, 입고배분, 비용)
- 계좌 입출금 기록 (엑셀업로드, 계정적용, 적요)

---

## 연동 API
- KAMIS API (시세 조회)
- 사방넷 API (판매가 전송)
- 크롤링/스크래핑 (경쟁사 판매가)
- 카카오톡 API (견적서 송신)
