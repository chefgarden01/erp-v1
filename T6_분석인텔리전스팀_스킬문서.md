# T6. 분석인텔리전스팀 스킬 문서

## 팀 역할: PDCA - Check (점검)

## 담당 모듈
- 분석관리1 (내부 영업 분석)
- 분석관리2 (의사결정 지원)
- AI 모델 학습 관리

---

## 필수 스킬

### 1. 내부 영업 분석 (리포트 생성)
- **입고량 분석**: 기간별, 마스터상품별, 상위품목, 매입가 이력
- **주문(출고)량 분석**: 기간별, 상품별, SKU별, 평균판매가
- **선별작업 기량 분석**: 입고량 - 재고량 = 사용량 / 주문량 = 폐기율
- **관리누수 폐기량 분석**: 입고이력코드 기준, 폐기량/일자/원인
- **매출이익관리**: 전체, 상품별, SKU별, 채널별
- **매출이익 할당관리**: 온/오프라인 사업부 매입비용 배분
- **비용분석**: 사업부별, 출고존별, 판매처별, 비용품목별

### 2. 비용절감
- 비용절감 시뮬레이션 (구매가, 사용량, 타업체 납품가, 절감액)
- 비용절감 제안보고서 생성

### 3. 의사결정 지원
- 실시간 시세 분석 조회
- KAMIS 가락도매시장 시세 조회
- 지정품목 시세 등락 추적
- 수입업체 수입현황 데이터 수집
- 뉴스/온라인 자동검색 → 리포트 생성 (지정/요청 키워드)
- 방송노출/검색량 증가 농산물 트렌드 자동추천 보고서

### 4. AI 학습 관리
- 외부 API 데이터 기반 AI 학습
- 자체 데이터 기반 추가 학습
- 학습 모델 성능 모니터링

### 5. 고급 분석 (추후 업데이트)
- 시세급등락 자동추적 보고서 (조건 입력 기반)
- 경영지표 컨설팅 보고서 (맥킨지 스타일)

---

## 연동 API
- KAMIS API (시세 분석)
- 뉴스/검색 크롤링 API
- AI 학습 엔진 API
- 수입현황 데이터 API

## 산출물
- 영업 분석 리포트 (입고/출고/폐기/매출이익/비용)
- 비용절감 제안 보고서
- 의사결정 지원 리포트
- 트렌드 추천 보고서
- 경영 컨설팅 보고서
