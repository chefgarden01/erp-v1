# T7. 운영지원팀 스킬 문서

## 팀 역할: PDCA - Do (실행)

## 담당 모듈
- 근태관리
- 3정5S 기업품격관리
- HACCP 상품관리
- 물류관리
- 고객클레임관리

---

## 필수 스킬

### 1. 근태관리 (한정 관리자 열람)
- 연차신청/조회 (모든 사용자)
- 특이사항 보고
- 작업배정 지시 (출근시간, 배정업무, 그룹별/직원별, 자동알림)
- 근무시간 등록 (지문인식 엑셀 자동등록)
- 현장직원 근무시간 조회 (전체, 직원별, 일별)
- 현장인건비 자동산출
- 사무실 급여지급 산출

### 2. 3정5S 기업품격관리
- TO DO LIST (항목입력, 관리주기, 담당자배정, 완료체크)
- 언어선택 기능 (AI 번역 연동)
- 경영관리/현장관리/판매관리 항목
- 담당자 지정 (입퇴사 시 변경, 자동알림)
- 이행사항 분석 리포트
- 경영철학/관리중점/회사방향/매뉴얼 등록

### 3. HACCP 상품관리
- 해썹 작업장 체크
- 관리 서류 자동생성 및 출력
- 생산 입력 (입고 → 상품생산 → 출고, 원산지)
- 수불부 양식 자동생성
- 작업장 TO DO LIST
- 유지보수 정보입력
- 이력조회 (청소, 유지보수, 생산, 입고, 출고)

### 4. 물류관리
- 자체 물류차량 등록 (보험기간, 보험료)
- 차량별 비용등록 (하이패스, 주유비, 수리비)
- 외부 물류차량 등록 및 비용 엑셀등록
- 온라인사업부 물류비용 등록

### 5. 고객클레임관리
- 클레임 등록 (사업부별, 주문번호별, 진행상태 메모)
- 클레임 비용 입력 (매출차감, 추가비용)
- 클레임 보고서 생성 및 담당자 자동전송
- 캄보디아어 자동번역 (AI API)
- 2회 이상 반복 SKU상품 자동모니터링 → 클레임 방지 리포트
- 클레임 분석 리포트 (채널별, 출고존별, 상품별, 비용현황)
