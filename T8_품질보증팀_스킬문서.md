# T8. 품질보증(QA)팀 스킬 문서

## 팀 역할: PDCA - Check (점검)

## 담당 모듈
- 전 모듈 기능 테스트
- 보안 검증
- 성능 검증

---

## 필수 스킬

### 1. 기능 테스트
- 모든 모듈 CRUD 기능 정상작동 검증
- 모듈 간 연동 기능 테스트
- 자동화 기능 테스트 (자동리포트, 자동알림, 자동코드생성)
- QR코드 인식/이동 테스트
- OCR 인식 정확도 테스트

### 2. 데이터 정합성 검증
- 모듈 간 데이터 흐름 일관성 확인
- 코드 체계 무결성 검증
- 입력/수정 이력관리 정확성 확인
- 재고 데이터 정합성 (입고-출고-재고 일치)
- 매출/매입 데이터 일치 검증

### 3. 보안 테스트
- 사용자 접근권한 테스트 (그룹별 접근 제한)
- 데이터 유출 방지 기능 검증
- 이력관리 추적 기능 검증
- 한정 관리자 메뉴 접근 차단 확인

### 4. 성능 테스트
- 대량 데이터 처리 속도 검증
- 동시 접속 부하 테스트
- API 응답 시간 측정
- 엑셀 대량 업로드 처리 시간

### 5. UI/UX 검증
- PC웹 브라우저 호환성 (크롬/엣지)
- 앱 버전 동작 확인
- 퀵메뉴 정상 동작
- 캄보디아어 병행표기 표시 확인
- 출력 양식 정상 출력 확인

### 6. API 연동 검증
- 외부 API 응답 정상 확인 (KAMIS, 카카오톡, OCR 등)
- API 장애 시 대체 처리 확인
- AI 모델 응답 품질 검증
