# 마스터상품 코드체계 정의서

## 1. 코드 형식

```
GTP-XX-YYYY

GTP   = 그트플 접두사 (고정)
XX    = 초성 그룹번호 (01~14)
YYYY  = 그룹 내 순번 (0001~9999)
```

## 3. 코드 생성 로직

### 3.1 자동생성 흐름

```
사용자가 상품명(한국어) 입력
    ↓
상품명 첫 글자에서 초성 추출
    ↓
초성 → 그룹번호 변환 (ㄱ→01, ㄴ→02, ...)
    ↓
해당 그룹의 기존 코드에서 최대 순번 조회
    ↓
최대 순번 + 1 → 새 코드 생성
    ↓
코드 필드에 실시간 표시
```

### 3.2 초성 추출 알고리즘

```
유니코드 한글 음절: U+AC00 ~ U+D7A3
초성 인덱스 = (charCode - 0xAC00) / (21 × 28)
19개 초성(ㄱㄲㄴㄷㄸㄹㅁㅂㅃㅅㅆㅇㅈㅉㅊㅋㅌㅍㅎ) → 14개 그룹으로 통합
쌍자음(ㄲ,ㄸ,ㅃ,ㅆ,ㅉ)은 기본 자음 그룹에 포함
```

### 3.3 순번 충돌 방지

- 저장 시점에 `generateMasterProductId()` 재호출하여 최신 순번 확인
- 같은 그룹 내에서만 순번 관리 (그룹 간 독립)
- 삭제(소프트)된 코드의 순번은 재사용하지 않음

## 4. 적용 시점

| 작업              | 코드 생성 방식                               |
|-------------------|----------------------------------------------|
| 신규등록 (모달)    | 상품명 입력 시 실시간 생성, 저장 시 최종 확정  |
| 엑셀 대량등록      | 각 행의 상품명(한국어)에서 자동 생성           |
| 수정               | 기존 코드 유지 (변경 불가, readonly)           |

## 5. 관련 함수

| 함수명                          | 위치      | 역할                                  |
|--------------------------------|-----------|---------------------------------------|
| `getChosungGroup(name)`        | app.js    | 상품명 → 초성 그룹번호(0~14) 반환      |
| `generateMasterProductId(name)`| app.js    | 상품명 → GTP-XX-YYYY 코드 생성         |
| `bindMasterProductIdAuto()`    | app.js    | 신규등록 폼에서 상품명↔코드 실시간 연동  |

## 6. 코드 정의 (codeDefinitions.js)

```javascript
master_product: {
    name: '마스터상품',
    prefix: 'GTP',
    fields: [
        {
            key: 'master_product_id',
            label: '마스터상품코드',
            type: 'text',
            required: true,
            auto: true,    // 자동생성 플래그
            hint: '상품명 초성 기반 자동부여 (GTP-[초성그룹]-순번)'
        },
        {
            key: 'product_name_kr',
            label: '상품명(한국어)',
            type: 'text',
            required: true  // 코드 생성의 기준이 되는 필수 필드
        },
        // ... 나머지 필드
    ]
}
```

## 7. 데이터 초기화

- 기존 순번 체계(GTP-0001~) 데이터 전체 삭제 완료
- 관련 변경이력(gtp_change_history_GTP-*) 초기화 완료
- 초기화 플래그: `localStorage.gtp_master_product_reset_v2 = 'done'`
- 브라우저 접속 시 1회 자동 실행, 이후 재실행 없음
