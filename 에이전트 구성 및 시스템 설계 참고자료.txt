###에이전트 구성시 참고
모든 모듈이 연동된다고 가정하고 에이전트팀 구성
스킬문서 작성
마크다운 파일 생성
에이전트 구성을 할 때 PDCA 워크플로우 방식 적용
외부 API 연동과 Ai 모델을 연동하는 시스템 구현
일부 모듈은 이미지와 API 외부 데이터를 통해 Ai 학습을 통한 리포트 생성, 자체 데이터를 통해 추가적인 학습

### 에이전트 구성시 참고할 우리회사 비지니스 모델 특징
관리자는 모두 한국인, 현장 생산자는 모두 외국인(캄보디아)
취급 상품이 많고 SKU 상품이 많음, 계속적인 확장 계획 있음
시세 변동이 아주 심하고 생육관리, 수급관리, 상품 품질 관리, 외국인의 특성에 따른 변수 요소가 많음
변동성과 변수가 많기 때문에 위기와 기회가 항상 상존
데이터분석을 통한 의사결정 구조 확립하면 이익을 극대화 할 수 있는 사업모델
농산물을 직접 생산하는 농업회사의 강점으로 온라인판매시장에서 주도권 형성 가능
현재 사업을 기반으로 추가적인 브랜치 사업 모델 가능성(치유돌봄사업, 가공사업, 수출사업 등)
