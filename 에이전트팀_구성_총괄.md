# 그트플 통합관리시스템 - 에이전트팀 구성 총괄

## 1. 시스템 개요

- **시스템명**: 그트플 통합관리시스템 (GTP Integrated Management System)
- **워크플로우**: PDCA (Plan-Do-Check-Act) 사이클 적용
- **대상**: 농업회사 통합 ERP 시스템
- **특징**: 외부 API 연동, AI 모델 학습 기반 리포트 생성, 다국어(한국어/캄보디아어) 지원

---

| 팀 번호 | 에이전트팀명 | 담당 모듈 | PDCA 역할 |
|---------|-------------|----------|----------|

| T2 | 농장관리팀 | 농장관리, 생육분석, 농장손익 | Do |

## 3. PDCA 워크플로우 매핑

```
[Plan] T1 시스템 아키텍처팀
  - 마스터코드 체계 설계, DB 스키마, API 연동 규격 수립
  - 전체 모듈 간 데이터 흐름 설계
       ↓
[Do] T2~T5, T7 실행팀
  - 각 업무 모듈별 기능 구현 및 데이터 처리
  - 외부 API 연동 (KAMIS, 카카오톡, OCR 등)
       ↓
[Check] T6 분석인텔리전스팀 + T8 QA팀
  - AI 기반 데이터 분석 및 리포트 생성
  - 시스템 테스트, 보안 검증, 성능 점검
       ↓
[Act] T9 통합관제팀
  - 분석 결과 기반 시스템 개선 조치
  - 알림/알람 자동 발송, 이상 징후 대응
  - 차기 PDCA 사이클 피드백 반영
```

## 4. 팀별 상세 역할

### T1. 시스템 아키텍처팀 (Plan)
- **마스터코드 자동생성** 엔진 설계 및 코드 배열 규칙 수립
- **QR코드 자동생성** 모듈 설계
- **사용자 계정/권한** 체계 설계 (경영그룹, 사용자그룹, 농장그룹, 물류현장그룹, 거래처그룹)
- **기초정보 코드** 체계 설계 (상품, 자재, 거래처, 비용, 농장, 직원, 물류, 배송타입 등)
- **UI/UX 설계** (PC웹/앱, 퀵메뉴, 아이콘 자동생성)
- **대량데이터 처리** 아키텍처 설계 (DB + 백엔드)
- **보안 정책** 수립 (데이터 유출 방지, 이력 추적)
- **출력 양식** 표준 설계

### T2. 농장관리팀 (Do)
- **하우스 현황** 종합 대시보드 구현
- **하우스 상세 관리** (작물등록, 기본값/변수값 입력)
- **수확관리** (수확기록, QR코드 자동입력, 잔량관리)
- **출고예약 및 확정** 처리
- **비용관리** (종자, 모종, 농자재, 수도광열비 등)
- **현장근무자 관리** (TO DO, 수확배정)
- **수확일정등록** 및 인원배정
- **시세 자동분석** 및 보고서 생성 (API 연동)
- **생산성 분석** (하우스별, 인원별, 작물별)
- **농장손익관리** (자동 손익보고서 생성)
- **AI 작물추천** (KAMIS API + AI 엔진)
- **작물 시뮬레이션** (손익 시뮬레이션, 실적 비교)
- **생육상태 자동분석** (이미지 업로드 → AI 분석 → 리포트)

### T3. 주문출고재고팀 (Do)
- **판매처별 주문수집** (엑셀, 수동, OCR, 중복방지)
- **주문집계 대시보드** 및 조회
- **쿠팡 주문리스트 엑셀 자동화**
- **주문현황 전체/상세 조회**, 시간대별 변동사항
- **추가주문 예상량 입력** 및 정확도 분석
- **상품재고등록** (QR, 입고이력코드, OCR, 수동)
- **생산입력** (SKU별 인력배정, 작업시간, 진도율)
- **부족/적정/과잉 재고 자동리포트** 및 알림
- **입고예약 요청** (긴급, 사전확보, 조건입고)
- **관심상품 시세조회** (KAMIS, 가락시장 API)
- **공급가 손익 시뮬레이션**
- **비상품 재고관리** (포장자재, 라벨지, 소모품 등)

### T4. 입고관리팀 (Do)
- **상품 입고**: 입고예약(발주), 검수등록(AI 품위분석), 반품등록, 입고확정(OCR/엑셀/건별)
- **입고이력코드 자동생성** (규칙: 입고일+매입처+상품명+일련번호)
- **입고내역 조회 및 이력관리**
- **입고담당자 지정/수정** (자동 알림)
- **매입확정 및 정산관리** (지급내역, 잔액, 계산서)
- **비상품 입고**: 발주 → 거래처 자동전송, 검수, 반품, 입고확정
- **비상품 입고이력코드 자동생성**

### T5. 판매관리팀 (Do)
- **오프라인 사업부**: 견적서 작성/공유, 시세조회, 판매손익 시뮬레이션, 계약확정, 거래처 자동송신
- **온라인 사업부**: 주문매출 조회, SKU별 매입가적용, 매출이익 조회
- **판매가 정보수집** 및 경쟁사 비교 (API/크롤링)
- **판매가 다운로드** → 사방넷 업로드 → 스토어 전송
- **판매손익 시뮬레이션** (마진설계 테이블)
- **손익 조회** (가마감/최종)
- **회계용 매출/매입 입력**
- **계좌 입출금 기록**

### T6. 분석인텔리전스팀 (Check)
- **내부 영업 분석**: 입고량, 주문량, 선별작업, 폐기량, 매출이익, 비용분석
- **비용절감 시뮬레이션** 및 제안보고서 생성
- **의사결정 지원**: 실시간 시세분석, KAMIS 연동, 수입현황 데이터 수집
- **뉴스/온라인 자동검색** → 리포트 생성
- **트렌드 자동추천 보고서** (방송노출, 검색량 증가 품목)
- **AI 모델 학습** 관리 (자체 데이터 기반 추가 학습)
- **시세급등락 자동추적** 보고서
- **경영지표 컨설팅 보고서** (맥킨지 스타일)

### T7. 운영지원팀 (Do)
- **근태관리**: 연차신청/조회, 작업배정, 근무시간, 인건비 산출, 급여산출
- **3정5S 관리**: TO DO LIST, 담당자 배정, 이행분석, 다국어 지원
- **HACCP 관리**: 위생관리 서류 자동생성, 생산입력, 수불부 자동생성
- **물류관리**: 차량등록, 비용등록(하이패스/주유/수리), 외부물류, 배송비
- **고객클레임관리**: 클레임등록, 비용입력, 보고서 생성(캄보디아어 번역), 반복클레임 자동모니터링, 분석리포트

### T8. 품질보증(QA)팀 (Check)
- **기능 테스트**: 전 모듈 기능 정상작동 검증
- **데이터 정합성 검증**: 모듈 간 데이터 흐름 일관성 확인
- **보안 테스트**: 접근권한, 데이터 유출 방지, 이력관리 검증
- **성능 테스트**: 대량데이터 처리 속도, 동시접속 부하
- **UI/UX 검증**: PC웹/앱 브라우저 호환성, 퀵메뉴 동작
- **API 연동 검증**: 외부 API 응답, 장애 시 대응 확인

### T9. 통합관제팀 (Act)
- **알림/알람 통합 관리** (문자, 카카오톡 API 연동)
- **시스템 모니터링** 및 이상 징후 감지
- **자동스케줄러 관리** (주문수집, 판매가수집, 시세추적)
- **모듈 간 연동 상태** 모니터링
- **PDCA 피드백** 반영 및 시스템 개선 조치
- **버전 업데이트** 관리 및 배포

        │                              │
        │                              ▼
